# COVID-19-Global-Data-Analysis

## Overview
Analysis of COVID-19 global data across 187 countries using Python and Pandas.

## Key Findings
- US had the most confirmed cases at 4,290,259
- UK had the highest death rate at 15.19% among top 10 affected countries
- Eastern Mediterranean had the best recovery rate at 80.59%
- Overall global death rate was 3.97% across 16.4 million cases

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Dataset
COVID-19 Country Wise Latest Data from KaggleAnalysis of COVID-19 global data across 187 countries using Python and Pandas. Exploring confirmed cases, death rates, recovery rates and WHO regional trends
